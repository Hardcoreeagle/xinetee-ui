# **App Name**: Xietee

## Core Features:

- Wallet Connection: Enable users to connect their wallets (MetaMask, WalletConnect) for secure access.
- Product Registration: Allow suppliers/admins to register new products by uploading details and documents to IPFS and storing the CID on the blockchain.
- Product Dashboard: Display a list of registered products with live updates fetched from the blockchain and backend cache.
- Product Detail View: Show detailed product information, including metadata, checkpoints, and IPFS document links.
- Checkpoint Management: Enable logistics/warehouse roles to add product checkpoints, updating the product's route map on the blockchain.
- Product Verification: Allow public verification of product authenticity via QR scan, displaying verification status based on blockchain data.
- User Feedback Collection: Collect user feedback via a form and submit it to the backend API.
- Admin Page: For internal admin use to manage roles and permissions. Assign/revoke roles and display wallet addresses and access levels.
- Error / Fallback Page: Fallback route for invalid URLs or failed product verification. Display a simple error message with an option to go back to the Home or Dashboard.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to evoke trust and reliability.
- Background color: Very light blue (#E8F0FE), almost white, for a clean and professional look.
- Accent color: A vibrant violet (#7952B3) will highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, which offers a modern, machined, objective, neutral look; suitable for both headlines and body text.
- Use simple, professional icons that are intuitive and easy to understand, focusing on clarity and efficiency.
- Design a clean and intuitive layout, emphasizing easy navigation and scannability for business users.
- Incorporate subtle animations to enhance user interaction without being distracting, maintaining a professional and trustworthy feel.
